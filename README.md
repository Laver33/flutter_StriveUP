Название StriveUP 

Описание: 
- Приложение с занятиями для улучшения вашей физической подготовки

Главные возможности:
 - Выбор нужной вам категории тренировок
 - Отслеживание изменения физической формы
 - Просмотр понятных и качественных упражнений / тренировок
 - Умный ИИ-помошник в чате
 - Калькулятор и трекер показателей
